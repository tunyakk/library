# Library (Библиотека)

Система управления библиотекой - десктопное приложение на C# / .NET 10 / WinForms.

Library(Библиотека) - производственно-практический проект автоматизации работы библиотеки: учёт книг, читателей, авторов, жанров, издательств, выдача и возврат книг, формирование отчётов.

### Что может проект

- Управление каталогом книг (добавление, редактирование, удаление, поиск)
- Учёт читателей и контактных данных
- Учёт авторов, жанров и издательств
- Выдача и возврат книг с отслеживанием статуса
- Отчёты и аналитика (графики выдач, статистика)
- Валидация входных данных
- Локальное хранилище SQLite - не требует отдельного сервера БД
 
## На чем создан проект

- Язык: C# 
- Платформа: .NET 10
- Интерфейс: Windows Forms
